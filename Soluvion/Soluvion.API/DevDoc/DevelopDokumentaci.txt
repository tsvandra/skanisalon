PROJEKT: Soluvion (Skani Salon) - SaaS Fodrászat Kezelő Rendszer

CÉL:
Egy .NET backend alapú, Vue.js frontenddel rendelkező webalkalmazás fejlesztése.
Jelenlegi fázis: Egyedi bemutatkozó oldal + Admin funkciók (CMS) egy adott szalonnak.
Jövőbeli cél: Multi-tenant (SaaS) rendszer, ahol több szalon is használhatja az appot saját adatbázis adatokkal.

TECHNOLÓGIAI STACK:
- Backend: .NET 10 Web API, Entity Framework Core, SQL Server. (Ma 2026.01.27.-e van, szoval letezik a .NET10, nem hazudok)
- Frontend: Vue 3 (Composition API), Vite, PrimeVue komponensek.
- Autentikáció: JWT Token (Bearer auth).

ARCHITEKTÚRA SZABÁLYOK (SaaS Előkészítés):
1. Adatbázis: Minden üzleti entitás (Service, Gallery, User, GalleryCategory) rendelkezik 'CompanyId'-vel.
2. Frontend: Nem drótozunk be fix cégneveket/logókat/színeket/elérhetőséget. Minden adatot API-ból kérünk le induláskor (`App.vue` -> `fetchCompanyData`).
3. Biztonság: Műveleteknél (POST, PUT, DELETE) a 'CompanyId'-t a User Token-ből (Claims) olvassuk ki a Backend oldalon.

EDDIG ELKÉSZÜLT FUNKCIÓK:
1. Publikus Oldalak:
   - Home: Bemutatkozás, Árlista gomb (Dinamikus színekkel), Időpontfoglalás gomb a kapcsolat oldara iranyitva.
   - Services (Árlista): Adatbázisból töltött árak.
   - Gallery: Kategóriákra szűrt képek.
   - Contact: Teljesen dinamikus. A cím, telefon, email, nyitvatartás szövegek, 1x social link (bovitesre vár) és Google Maps embed az SQL-ből jön.

2. Admin Funkciók (Csak bejelentkezve):
   - Login: JWT token alapú, biztonságos belépés.
   - Galéria Kezelés:
     - Kép feltöltése.
     - Új kategória létrehozása ("+ Új kategória..." opció).
     - Kép törlése (saját céghez kötve).
   - Árlista Kezelés: Új szolgáltatás felvétele, törlése.
   - Cégbeállítások (Settings):
     - Külön Admin oldal (`/beallitasok`).
     - Szerkeszthető: Alapadatok: Cegnev, Emailcim, telefonszam, Cím, Nyitvatartás szövegek (HTML támogatással)(4x textbox), Social linkek.
     - Szerkeszthető Dizájn: Primary/Secondary színek valós idejű módosítása.

3. Backend API Végpontok:
   - Auth (Login/Register).
   - Company (GetById, Update - PUT, CreateCompany).
   - Gallery (Get, GetCategories, Upload, Delete).
   - Service (Get, Post, Delete).

DESIGN SYSTEM (Szigorúan tartandó!):
- Technikai Megvalósítás: CSS Változókat (Variables) használunk (`var(--primary-color)`, `var(--secondary-color)`), amiket az `App.vue` állít be az adatbázis alapján.
- Alapszínek (Jelenlegi Skani beállítás):
  - Másodlagos (Háttér/Szöveg): #1a1a1a (Fekete/Sötétszürke).
  - Elsődleges (Akció/Kiemelés): #d4af37 (Arany).
  - Szöveg kontraszt: Sötét háttéren világos szöveg.
- Brand Szín (Logó):
  - Kék Átmenet: rgb(24, 160, 198) -> rgb(17, 108, 135).
  - HASZNÁLAT: Csak a Logóban! A UI elemek (gombok, ikonok) maradjanak az Elsődleges színnél (Arany).
- Stílus:
  - Elegáns, minimalista. Font: 'Playfair Display'.
  - Gombok: Arany háttér, Fekete szöveg, Keret nélkül.
  - Inputok: Fókuszban Arany keret.
